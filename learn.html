Прохождение курса по использованию Git от Geekbrains!

Резюме второго урока.

Clone - копирование удаленного репозитория.

Commit - сохранение изменений в локальном репозитории.

Push - передача изменений в удаленный репозиторий.

Pull - получение изменений из удаленного репозитория.

Резюме третьего урока.

Delete - удаляет файл целиком.

Discard - отменить все изменения в выбраном файле. До выполнения commit.

Revert - отменяет изменения после commit.

Reset - отмена последних изменений. До выполнения push.

в SGit отсутствуют команды Disckard и Revert.

Резюме четвертого урока.

Branch - создает новую ветку.

Checkout - переключение между ветками.

Merge - производит слияние веток.

SGit поддерживает передачу изменений только в ветку master. В SGit отсутствует возможность удалять отдельную ветку.
